Description

I have chosen Option B: Cartoon Interpolation as my project. I will attempt to implement the continuous deformation of cartoons/animations using the polygon method, and attempt to extend this to potentially manipulate images that are not strictly SVG files. An example of what I hope to achieve can be found here, which achieves some of its animated effects by warping and deforming a drawn image in a way presumably similar to what has been described.

Concrete Steps

  1. Modify the project to be able to open/render non-SVG files along with an SVG file to sit on top of it and act as the frame/skeleton.
  2. Implement method to subdivide given polygon into triangles.
  3. Linearly interpolate these triangles in order to manipulate the image below.
  4. If time allows, implement methods for point and skeleton based manipulation, as mentioned here here.
  5. Record a video of the finished product in action.
