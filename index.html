Write Up

Name

Sophia Zhu

Description

My original project choice was Option B. However, while attempting to implement it, I ran into many problems (caused mainly by my lack of experience in OpenGL and C++). After spending a day trying to parse the starter code, failing to properly integrate my planned changes to it (for example, the starter code appears to render all the characters each time it updates, which works for simple shapes but ends up effectively freezing my computer due to lag on textures), and having done little properly graphics-related coding, I pragmatically decided it would be a better idea to switch to Option I. For Option I, I basically wrote code to demosaic the Bayer mosaic, account for dead/flawed pixels and input, and autofocus the image, as described in the project description.

Instructions

The project should be compiled and run the way described on the project site, found here. I've also added an input option for a calibration file, so running the program would be: 

./PROGRAMNAME -noiselevel 1 -verybadsensor -calibrate PATH_TO_SCENES/calibrate.bmp PATH_TO_SCENES/SCENE.bin OUTPUT.bmp

Proposal

Name

Sophia Zhu

Description

I have chosen Option B: Cartoon Interpolation as my project. I will attempt to implement the continuous deformation of cartoons/animations using the polygon method, and attempt to extend this to potentially manipulate images that are not strictly SVG files. An example of what I hope to achieve can be found here, which achieves some of its animated effects by warping and deforming a drawn image in a way presumably similar to what has been described.

Concrete Steps

  1. Modify the project to be able to open/render non-SVG files along with an SVG file to sit on top of it and act as the frame/skeleton.
  2. Implement method to subdivide given polygon into triangles.
  3. Linearly interpolate these triangles in order to manipulate the image below.
  4. If time allows, implement methods for point and skeleton based manipulation, as mentioned here here.
  5. Record a video of the finished product in action.
